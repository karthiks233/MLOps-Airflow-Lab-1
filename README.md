# MLOps-Airflow-Lab-1


I have used a different clustering algorithm - DBScan and Standard scaler

##Steps to Run:
1. Clone the entire Repo
2. Run Docker Engine
3. Within the project -> Run docker compose up --build
4. Sign in to Airflow using: username and password: airflow
5. To close the project -> Run docker compose down
